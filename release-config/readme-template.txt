Onboard.qs — Interactive Onboarding Tours for Qlik Sense
=========================================================

Version: __VERSION__

Getting Started
---------------

1. Upload the included "onboard-qs.zip" extension file to Qlik Sense:

   - Qlik Cloud:
     Open Management Console > Extensions > Add (+) > Upload the zip file.

   - Qlik Sense Enterprise on Windows (client-managed):
     Open QMC > Extensions > Import > Select the zip file.

2. Reload any open app editor pages in your browser.
   The extension will only appear in the assets panel after a page reload.

3. Drag the "Onboard.qs" visualization onto a sheet and configure your
   onboarding tours in the property panel.
